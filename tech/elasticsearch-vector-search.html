Elasticsearch: Vector Search in Plain English

Vector search answers “things like this” instead of “things that match these exact words.”

What you get

  • Semantic search (meaning-based)
  • Hybrid search (keyword + vector)
  • Better retrieval for RAG-style workflows

What to watch

  • Vectors cost disk and memory
  • Index/query settings trade speed vs accuracy
  • Benchmark with real queries, not vibes

A simple evaluation loop

  • Pick 20 real queries
  • Define “good results”
  • Compare keyword-only vs hybrid vs vector-only
